# Consistent hashing(Hash ring)

## 기본 hash ring
- hash 함수로 나온 값의 결과를 사용
- 노드 별 해시링의 범위를 최대한 균일하게 나눠서 지정
- hash 결과를 가장 가까운 노드에 배정
- 문제는 하나의 노드가 아웃되는 경우, 트래픽이 다른 노드로 집중되며 순차적으로 노드가 아웃되는 현상이 발생할 수 있음

## vnode를 사용하는 hash ring
- 각 노드별 가상 노드를 추가하여 모든 링에 더 촘촘하게 노드가 존재하여 아웃되는 경우 트래픽을 더 균일하게 분배가능
